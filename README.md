# 🪙 Wallet Project Version 1.0

A simple e-wallet application

---

## 🔧 Quick Start

```bash
# Start the server
cd server
npm install
touch .env

add 
DB_URI=
JWT_SECRET=
in .env

npm run dev   # Runs on http://localhost:3400

# Start the client
cd client
npm install
npm run dev  

## ⚙️ Tech Stack
- Node.js + Express (Backend)
- React (Frontend)
